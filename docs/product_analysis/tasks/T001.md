## 📝 NOTAS TÉCNICAS

### 1. Sobre `better-sqlite3`

O `better-sqlite3` está no `package.json` mas **não é necessário** para o MVP que usa PostgreSQL. Possíveis usos:
- Testes unitários com banco SQLite em memória (rápido)
- Desenvolvimento local sem PostgreSQL

**Recomendação:** Manter apenas se for usar para testes. Caso contrário, remover:
```bash
npm uninstall better-sqlite3
```

### 2. Sobre HMR (Hot Module Replacement)

O `--hmr` flag no `node ace serve --hmr` já está configurado. Isso permite:
- Hot reload de controllers e middleware
- Hot reload de templates Edge
- Hot reload de assets via Vite

### 3. Sobre TypeScript

O projeto já está configurado com TypeScript. Verificar `tsconfig.json`:
```json
{
  "extends": "@adonisjs/tsconfig/tsconfig.app.json",
  "compilerOptions": {
    "rootDir": "./",
    "outDir": "./build"
  }
}
```

### 4. Sobre Import Aliases

O `package.json` já tem import aliases configurados:
```json
{
  "imports": {
    "#controllers/*": "./app/controllers/*.js",
    "#services/*": "./app/services/*.js",
    "#models/*": "./app/models/*.js",
    "#middleware/*": "./app/middleware/*.js",
    "#validators/*": "./app/validators/*.js",
    "#config/*": "./config/*.js",
    "#start/*": "./start/*.js"
  }
}
```

Uso no código:
```ts
import AuthService from '#services/auth/auth_service'
import User from '#models/user'
```
